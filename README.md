## Chip8 Interpreter Core

A Chip-8 Interpreter Core written in C.

---

Undefined platform dependant functions you need to implement for your target platform. 
 - chip8_beep(CHIP8*);
 - chip8_random(CHIP8*);
 - chip8_render(CHIP8*);

See my SDL2 or Arduino Implementations for an example.
